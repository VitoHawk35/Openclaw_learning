# 用户偏好设置

## 沟通偏好
- **时间预估**: 必须在执行前告知预计耗时
- **探索类任务**: 先给结论，再问是否需要深入
- **等待容忍度**: 低，需要快速反馈

## 学习风格偏好
- **学习方式**: 两遍学习法
  - 第一遍：快速建立全局知识体系（是什么、为什么、技术演进）
  - 第二遍：深度学习细节
- **切入点**: 从"问题→解决方案"角度理解技术
- **避免**: 一上来死扣细节，不知道用途

## 教学角色定位
- **角色**: 拥有教学大纲的教师
- **方法**: 基于6.0资料和视频大纲进行复现式教授
- **重点**: 先讲技术出现的背景和解决的问题，再讲实现

## 记忆管理
- **每日记忆文件**: `memory/YYYY-MM-DD.md`
- **学习档案**: `memory/study-plan.md`
- **本文件**: `memory/user-preferences.md`
